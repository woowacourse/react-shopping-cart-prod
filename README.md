## 기능 목록

- 사용자별 장바구니 기능 동작
- 장바구니에 상품 추가
- 장바구니 상품 조회
- 장바구니 상품 주문 / 쿠폰 적용
- 장바구니 상품 삭제
- 장바구니 상품 수량 변경
- 주문 목록 조회
- 주문 상세 내역 조회

<br>

## 실행 방법

1. 레포지토리 클론

```bash
git clone -b leejin-yang --single-branch https://github.com/woowacourse/react-shopping-cart-prod.git
```

2. 의존성 설치

```bash
npm install
```

3. 실행

```bash
npm run start
```
