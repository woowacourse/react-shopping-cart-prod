## 🚀 실행 방법

1. 프로젝트를 클론해줍니다.

```
git clone "프로젝트.git"
```

2. 브런치를 이동해줍니다.

```
git checkout step2
```

3. 패키지를 설치해줍니다.

```
yarn install
```

4. 실행 스크립트를 터미널에 입력합니다.

```
yarn run dev
```
